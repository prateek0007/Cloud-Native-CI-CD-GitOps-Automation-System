#  DevOps Portfolio – End-to-End Projects

This repository contains hands-on DevOps projects covering CI/CD, Kubernetes, Monitoring, Self-Healing Systems, and Cloud (GitOps).

The goal of this portfolio is to demonstrate real-world DevOps practices by building production-like systems.

---

#  Projects Overview

## 🔹 Project 1: CI/CD Pipeline with Kubernetes

### What I built
- Containerized application using Docker  
- CI/CD pipeline using GitHub Actions  
- Kubernetes deployment with scaling  
- Rolling updates with zero downtime  
- Monitoring using Prometheus & Grafana  

### Outcome
- Automated build and deployment pipeline  
- Scalable and production-ready application deployment  
- Real-time monitoring of system performance  

---

## 🔹 Project 2: Self-Healing DevOps System (SRE)

### What I built
- Kubernetes deployment with health checks  
- Liveness & Readiness probes  
- Prometheus alert rules  
- Alertmanager integration  
- Auto-healing script for recovery  

### Outcome
- System can detect failures automatically  
- Alerts are generated for system issues  
- Automated recovery reduces manual intervention  
- Improved reliability and uptime  

---

## 🔹 Project 3: Cloud-Native GitOps Deployment (AWS EKS + ArgoCD)

### What I am building
- Kubernetes cluster on AWS (EKS)  
- GitOps workflow using ArgoCD  
- CI pipeline using GitHub Actions  
- Automated deployment from Git to Kubernetes  

### Expected Outcome
- Fully automated deployment pipeline (Git → Cluster)  
- Cloud-native scalable infrastructure  
- Declarative and version-controlled deployments
